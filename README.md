# The Biological Vault - SDG 14

**Version:** 1.0
**Date:** 25/03/2026

## Usage Notes
This interactive generative artwork abstracts a coral reef into a "Biological Database" to address SDG 14 (Life Below Water).
- **Interaction:** Simply move your mouse cursor across the canvas to interact with the artwork. 
- **Effect:** Your cursor acts as a "Data Corruptor" (a virus scanner). Hovering over the healthy, flowing ASCII DNA streams will cause them to violently glitch, turning into dead binary codes and flashing red error symbols. This visualizes the devastating and immediate impact of human interference on fragile marine ecosystems.

## Installation Instructions
1. Unzip the project folder (if compressed).
2. Users should start a local web server in the root directory to run the project. 
   - *If using VS Code:* Install the "Live Server" extension, right-click on `index.html`, and select "Open with Live Server".
   - *If using Python:* Open the terminal in the project directory, type `python -m http.server 8000`, and then access `http://localhost:8000` in your web browser.
3. Access the `index.html` file to see and interact with the artwork in full-screen.
No additional dependencies or plug-in installations are required (the p5.js library is loaded via CDN).
